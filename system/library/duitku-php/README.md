Duitku-PHP
===============

Duitku :heart: PHP!

This is the all new PHP client library for Duitku 1.3.0. This is the official PHP wrapper for Duitku Payment API. Visit [https://www.duitku.com] for more information about the product and see documentation at (http://docs.duitku.com/) for more technical details.

## Installation

### Composer Installation

If you are using [Composer](https://getcomposer.org), add this require line to your `composer.json` file:

```json
{
	"require": {
		"duitku/duitku-php": "master"
	}
}
```

and run `composer install` on your terminal.

### Manual Instalation

If you are not using Composer, you can clone or [download](http://docs.duitku.com/) this repository.
